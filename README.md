<h1 align="center">Random Addition Program (C++)</h1>

<p align="center">
A simple C++ program that generates two random numbers and displays their sum in a formatted way.
</p>

---

## 📌 Features

- Generates random numbers using `rand()`  
- Displays a formatted addition problem  
- Calculates and shows the result  
- Demonstrates basic use of:
  - `cstdlib`
  - `ctime`
  - formatted output  

---

## 🛠️ Technologies Used

- C++  
- Standard Libraries (`iostream`, `cstdlib`, `ctime`)
